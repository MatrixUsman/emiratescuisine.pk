# Logo Setup Instructions

## Logo Location

Place your logo image at: `public/logo/logo.png`

## Logo Requirements

- **Format**: PNG (with transparency) or SVG
- **Size**: Recommended 200x200px or larger
- **Background**: Transparent preferred
- **Aspect Ratio**: Square (1:1) works best

## Current Logo Description

Based on your logo:
- Arabic calligraphy at the top (stylized black graphic)
- "emirates cuisine" text below in lowercase
- Black text on white/transparent background

## How to Add Logo

1. Save your logo file as `logo.png`
2. Place it in `public/logo/` folder
3. The header will automatically display it

## Alternative Formats

If you have the logo in a different format:
- SVG: Save as `logo.svg` and update Header.tsx to use `.svg`
- JPG: Save as `logo.jpg` and update Header.tsx to use `.jpg`

The logo will appear in the header next to the "emirates cuisine" text.
